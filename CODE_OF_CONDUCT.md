# Code of Conduct

## Our Standards

- Be helpful and constructive
- Assume good intent
- Respect differing viewpoints and experience levels
- Give and accept feedback gracefully
- Focus on what's best for the project and community

## Unacceptable Behavior

- Harassment, trolling, or personal attacks
- Publishing others' private information
- Deliberately destructive behavior toward the project

## Enforcement

Maintainers will remove, edit, or reject contributions that violate this code. Repeated violations result in a ban.

Report issues to conduct@getkoji.dev.
